## SECCIÓN 6: PREGUNTAS FRECUENTES

**¿Necesito saber programar?**
No. El curso está diseñado para personas sin conocimientos técnicos. Si sabés usar Google Sheets, ya sabés usar el sistema.

**¿Funciona sin internet?**
El servidor corre en tu celular. Para recibir mensajes de WhatsApp/Instagram necesitás internet, pero la base de datos local funciona sin conexión.

**¿Cuánto consume de batería?**
Muy poco. Termux consume menos que WhatsApp. En un celular con batería decente, dura todo el día.

**¿Puedo usarlo con más de una posada?**
Sí. Cada posada es una configuración separada. Podés tener múltiples CRM en el mismo celular.

**¿Qué pasa si se me apaga el celular?**
El respaldo automático en GitHub te protege. Cuando volvás a encender, todo se restaura.

**¿Hay actualizaciones?**
Sí. V2 incluye actualizaciones gratuitas por 1 año. Los módulos base se actualizan cuando Meta o Google cambian sus APIs.
